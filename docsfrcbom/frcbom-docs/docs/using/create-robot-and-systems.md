### Create a robot

From the **Admin** home page, click **Create Robot**, provide a name (e.g., *Reefscape 2025*), and save.

### Default systems

FRCBOM creates **5 default systems**. You can rename them:
- To change the system's name, **hover over its name** (like Google Docs), and rename.

### Fetching a system (what you need)

To fetch data for a system, provide:

1. **Main assembly URL** of the system (Onshape link).  
2. **Custom subassemblies** (links) that belong to this system.  
   - Do **not** include COTS swerve or other vendor assemblies.  
3. **All relevant Part Studios** for the system — required for:
   - **Downloading the part** from the site
   - **Viewing** it in the **3D viewer**

> You will need a valid **Onshape API key** (If you dont have please refer to [this page](../setup/fs-and-settings#create-your-onshape-api-keys))
