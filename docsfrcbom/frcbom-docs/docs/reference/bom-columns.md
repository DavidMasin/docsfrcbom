### Typical columns
- **Part Name** — human‑readable name
- **Quantity** — how many are needed
- **Material** — material/stock for in‑house parts
- **Description** — any helpful notes
- **Pre Process / Process 1 / Process 2** — which **machine** or action is needed
- **Available qty** — for COTS, how many you already have

> Admins map machine names to export **CAD formats** (e.g., Router→DXF, Mill→STEP).
