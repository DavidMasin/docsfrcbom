### Card colors
- **Green** — finished (all required steps complete OR enough COTS on hand)
- **Yellow** — partially complete
- **White** — not started

### Current process selection
1. If **Pre‑Process** exists and is not complete → it’s current.
2. Else if **Process 1** exists and not complete → it’s current.
3. Else if **Process 2** exists and not complete → it’s current.
4. Else → **Completed**.
