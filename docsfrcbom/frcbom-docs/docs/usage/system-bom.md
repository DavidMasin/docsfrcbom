### System BOM (the workhorse)
Each card = one BOM item.

**At a glance**:
- **Quantity Needed**
- **Material** and **Description**
- **Current Process** (what should happen next)
- **Status color**: green (done), yellow (in‑progress/partial), white (not started)

**Actions**:
- 👁️ **Preview** a part in 3D
- ⬇️ **Download CAD** for the next/current process (format auto‑picks based on machine)
- ✎ **Update counts** for Pre‑Process / Process 1 / Process 2 (admins or members, per team policy)
