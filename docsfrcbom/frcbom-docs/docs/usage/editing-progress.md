### Editing progress
For in‑house parts, each card can track 3 steps:
- **Pre‑Process** (e.g., cut blanks)
- **Process 1** (e.g., router/mill/lathe)
- **Process 2** (optional finishing step)

Enter how many are **done** in each step. When a step reaches the **quantity**, the next step becomes the **current process** automatically. Completed parts turn **green**.
