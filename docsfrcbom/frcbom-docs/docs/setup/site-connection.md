title: Connect Onshape to FRCBOM
---

- **BOM Source**: Provide the BOM URL (or upload/export if your workflow prefers CSV/XLSX).
- **GLTF Models**: For 3D previews, provide GLTF model URLs per part or system.
- **Subassemblies**: Add subassembly URLs; FRCBOM merges their parts into the main system automatically.
- **Live Updates**: Quantity edits propagate in real time to all users.
