### Link Onshape (optional but recommended)
Linking Onshape enables: **3D previews**, **per‑part downloads**, and **assembly exports**.

1. Open a **System** → **⚙️ Edit System Settings**.
2. Paste your **Assembly URL** (and any **Sub-assembly URLs**, optional).
3. (Optional) Add **Part Studio URLs** used for in‑house parts.
4. Save, then click **🔄 Fetch BOM**. The system’s parts will appear on the BOM.
5. Return to the **System BOM** to see cards for each part.
