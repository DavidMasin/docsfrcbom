### What you’ll learn
- Create or join your team
- Create your first robot and systems
- (Optional) Link Onshape for 3D previews and CAD downloads
- Define machines so the site knows which CAD formats to export

### Requirements
- A mentor/lead to own the initial setup
- Team number and a shared team password
- (Optional) An Onshape document that contains your assemblies/part studios
