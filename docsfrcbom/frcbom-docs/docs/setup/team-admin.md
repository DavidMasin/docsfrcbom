title: Team Admin
---

- **Create Team**: Provide team number and name, then set **Admin** and **User** passwords.
- **Materials**: Define as JSON (name, color/notes optional).
- **Machines**: Define as JSON including **stage** (`Process` or `PreProcess`) and optional `cad_format`.
- **Credentials**: You can change team name and passwords later in **Credentials**.
