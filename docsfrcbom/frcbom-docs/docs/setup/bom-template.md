---
title: Create the "FRCBOM" BOM Template
sidebar_position: 3
---

In Onshape, create a **new BOM template** named **FRCBOM** with the following columns, in this order:

1. **Item**  
2. **Name**  
3. **Description**  
4. **Quantity**  
5. **Material**  
6. **Mass**  
7. **FRCBOM - Bom Material**  
8. **FRCBOM - Process 1**  
9. **FRCBOM - Process 2**  
10. **FRCBOM - Pre Process**

After creating the template, open any assembly you want to use with FRCBOM and click **Apply Template** to apply **FRCBOM** to its BOM.
