title: Install the FeatureScript
---

1. In **FRCBOM → Team Settings**, paste your FeatureStudio URL.
2. Use **Open FS** to verify, and **Copy URL** to share with teammates.
3. In Onshape, import or create your FeatureStudio and include any team‑specific metadata helpers.

> The FeatureScript is optional but recommended to auto‑fill properties or enforce naming conventions.
