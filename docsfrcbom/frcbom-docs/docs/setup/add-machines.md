### Define machines & formats
Machines tell frcbom.com which **CAD format** to export for each process.

Common examples:
- **Router** → `DXF`
- **Mill** → `STEP`
- **Lathe** → `STEP`
- **Laser** → `DXF`
- **3D Printer** → `STL`

> You can filter parts by machine later and download everything that machine needs.
