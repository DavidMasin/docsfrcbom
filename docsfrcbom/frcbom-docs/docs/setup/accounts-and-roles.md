## Create your team account

1. Go to **https://frcbom.com/register**.
2. Enter your **Team Number** and **Team Name**.
3. Choose two passwords:
   - **Admin password** — for managing settings, machines, materials, FeatureScript, robots/systems, and integrations.
   - **User password** — for day‑to‑day usage (updating quantities, marking steps complete, downloading CAD, etc.).

## Admin vs User

| Capability | Admin | User |
|---|:---:|:---:|
| Change team name/passwords | ✅ | ❌ |
| Configure **Machines** & **Materials** | ✅ | ❌ |
| Set FeatureScript URL & property IDs | ✅ | ❌ |
| Create robots/systems | ✅ | ❌ |
| Edit BOM fields/status in UI | ✅ | ✅ |
| Download CAD, use viewers | ✅ | ✅ |

Use the **Admin** account for configuration tasks; give **User** access to students/mentors for manufacturing workflow.
