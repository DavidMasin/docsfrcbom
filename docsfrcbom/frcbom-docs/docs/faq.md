## Frequently Asked Questions

### Who are these docs for?
Students, mentors, and anyone using **frcbom.com**—no coding required.

### Do I need an Onshape account?
Only if your team links Onshape for CAD previews/downloads. See **Setup → Link Onshape**.

### Can non-admins mark parts as complete?
Yes, if your team lead enables it in System Settings. See **Using the Site → Editing Progress**.
